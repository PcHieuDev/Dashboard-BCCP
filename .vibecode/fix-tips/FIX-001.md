## SOURCE

- Related REQ: REQ-018

## ISSUE DESCRIPTION
TIP-002 đã sửa `ten_Cum` → `ten_cum` trong `analytics/global_metrics.py` và `analytics/revenue.py` (do Chủ thầu patch trong VERIFY).
Tuy nhiên còn tồn tại `ten_Cum` (chữ C hoa) trong **3 file callback** nữa:

| File | Số instance | Loại |
|------|------------|------|
| `dash_app/callbacks/global_callbacks.py` | 13 | SQL query + DataFrame column |
| `dash_app/callbacks/alerts_callbacks.py` | 6 | SQL query + DataFrame column |
| `dash_app/callbacks/service_callbacks.py` | 6 | SQL query + DataFrame column |

SQLite trả về cột `ten_cum` (viết thường) nên tất cả `r["ten_Cum"]` sẽ crash `KeyError`.

## ROOT CAUSE
TIP-002 chỉ fix `analytics/global_metrics.py` nhưng không scan đủ toàn bộ project.
Thợ thi công cần grep và fix tất cả instance `ten_Cum` trong thư mục `dash_app/callbacks/`.
