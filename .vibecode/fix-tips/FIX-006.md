## ISSUE DESCRIPTION

**Kỳ vọng:** Top 10 xã hoàn thành kế hoạch hiển thị đúng, cộng dồn actual từ bưu cục lên xã  
**Thực tế:** Kết quả rỗng hoặc tỷ lệ phi lý do:
1. Join trực tiếp `plans.ma_buu_cuc` (4 số) với `agg_monthly.buu_cuc` (6 số) → không khớp
2. Filter `nhom_dich_vu = 'BCCP'` trong `agg_monthly` → không tồn tại (thực tế là `'Truyền thống'`, `'TMĐT'`, `'Quốc tế'`)

## ROOT CAUSE (đã xác nhận từ DB)

```
plans BCCP: ma_buu_cuc = '460000' (6 số = ma_bc)
            nhom_dich_vu = 'BCCP', ten_dich_vu = 'Truyền thống'/'TMĐT'/'Quốc tế'

plans HCC:  ma_buu_cuc = '4670'   (4 số = ma_bdx)
            nhom_dich_vu = 'HCC', ten_dich_vu = 'Chuyển phát HCC'

agg_monthly.nhom_dich_vu: 'Truyền thống', 'TMĐT', 'Quốc tế', 'Hành chính công'
agg_monthly.buu_cuc:      '460000', '463800', '463890'... (6 số = ma_bc)

dim_buucuc: ma_bc='460000', ma_bdx='4600', ten_bdx='Bưu điện xã Diễn Châu', ten_cum='Diễn Châu'
```

**Hệ quả:** Khi "xem theo xã" trong Top 10, phải cộng dồn tất cả `ma_bc` có cùng `ma_bdx`:
- BCCP actual: SUM(agg_monthly) WHERE buu_cuc IN (460000, 463800, 463890...) = SUM cho xã 4600
- HCC actual: SUM(agg_monthly WHERE nhom='Hành chính công') GROUP BY ma_bdx (đã là cấp xã)
- BCCP plan: SUM(plans WHERE nhom='BCCP') WHERE ma_buu_cuc = '460000' (là ma_bc, group lại theo ma_bdx)
- HCC plan: SUM(plans WHERE nhom='HCC') WHERE ma_buu_cuc = '4670' (đã là ma_bdx)

### Logic đúng — compare_type='plan':

```python
# === BCCP: cộng dồn từ cấp bưu cục lên cấp xã ===
# Actual BCCP: nhom_dich_vu IN ('Truyền thống','TMĐT','Quốc tế')
# Group by ma_bdx qua JOIN dim_buucuc ON buu_cuc = ma_bc
SQL: SELECT b.ma_bdx, SUM(a.tong_doanh_thu) as dt_bccp
     FROM agg_monthly a
     JOIN dim_buucuc b ON a.buu_cuc = b.ma_bc
     WHERE a.nhom_dich_vu IN ('Truyền thống','TMĐT','Quốc tế')
       AND a.nam=? AND a.thang=?
     GROUP BY b.ma_bdx

# Plan BCCP: ma_buu_cuc LÀ ma_bc (6 số), group lên ma_bdx qua JOIN
SQL: SELECT b.ma_bdx, SUM(p.ke_hoach_doanh_thu) as plan_bccp
     FROM plans p
     JOIN dim_buucuc b ON p.ma_buu_cuc = b.ma_bc
     WHERE p.nhom_dich_vu = 'BCCP' AND p.nam=? AND p.thang=?
     GROUP BY b.ma_bdx

# === HCC: nạp ở cấp xã, so sánh thẳng ===
# Actual HCC: nhom_dich_vu = 'Hành chính công'
# Group by ma_bdx qua JOIN dim_buucuc (buu_cuc = ma_bc → ma_bdx)
SQL: SELECT b.ma_bdx, SUM(a.tong_doanh_thu) as dt_hcc
     FROM agg_monthly a
     JOIN dim_buucuc b ON a.buu_cuc = b.ma_bc
     WHERE a.nhom_dich_vu = 'Hành chính công'
       AND a.nam=? AND a.thang=?
     GROUP BY b.ma_bdx

# Plan HCC: ma_buu_cuc LÀ ma_bdx (4 số), dùng thẳng
SQL: SELECT ma_buu_cuc as ma_bdx, SUM(ke_hoach_doanh_thu) as plan_hcc
     FROM plans
     WHERE nhom_dich_vu = 'HCC' AND nam=? AND thang=?
     GROUP BY ma_buu_cuc

# === Gộp và tính tỷ lệ ===
# Merge tất cả theo ma_bdx (INNER JOIN để chỉ lấy xã có cả actual lẫn plan)
# actual_total = dt_bccp + dt_hcc
# plan_total = plan_bccp + plan_hcc
# ratio = actual_total / plan_total * 100.0
```
