## Phần 1: Area Chart — Luân chuyển KH theo 4-6 tháng

### Yêu cầu
Biểu đồ vùng chồng (Stacked Area) hiển thị tỷ trọng DT của 3 loại KH qua 4-6 tháng gần nhất.
Trả lời: "Tệp KHM tháng trước có chuyển hóa thành Hiện hữu tháng này không?"

### Logic data
```python
# Cho mỗi tháng T-5 → T hiện tại:
months_kh_data = []
for m in range(max(1, current_month - 5), current_month + 1):
    df = query_revenue(db, year, "month", month=m, cum=cum, ...)
    # classify_customers đã được gọi bên trong query_revenue
    dt_by_kh = df.groupby("loai_kh")["cuoc_tt_tong"].sum()
    months_kh_data.append({
        "Tháng": f"T{m:02d}",
        "Hiện hữu": dt_by_kh.get("Hiện hữu", 0),
        "KHM/Tái bán": dt_by_kh.get("KHM/Tái bán", 0),
        "Vãng lai": dt_by_kh.get("Vãng lai", 0)
    })
```

## Phần 2: Top 10 CMS đa chiều

### Yêu cầu
DataTable Top 10 khách hàng CMS theo doanh thu, có cột:
- `CMS` | `DT kỳ này` | `% Tăng trưởng` (so kỳ trước) | `Nhóm DV chính` | `Cảnh báo`

### Logic data
```python
# Query DT theo CMS kỳ hiện tại
df_cur_cms = query_revenue(db, ..., group_by_primary='cms')
# Query DT theo CMS kỳ trước
df_prev_cms = query_revenue(db, ..., prev period, group_by_primary='cms')

# Merge và tính % tăng trưởng
df_merge = df_cur_cms.merge(df_prev_cms, on='cms', suffixes=('', '_prev'), how='left')
df_merge['pct_change'] = (df_merge['cuoc_tt_tong'] - df_merge['cuoc_tt_tong_prev']) / df_merge['cuoc_tt_tong_prev'] * 100

# Nhóm DV chính = nhóm chiếm % cao nhất
# Query thêm group_by=['cms', 'nhom_dv'] → pivot → tìm max

# Cảnh báo: 🔴 nếu giảm >20% hoặc giảm liên tục 2 tháng
df_merge['canh_bao'] = df_merge['pct_change'].apply(lambda x: '🔴' if x < -20 else '')

# Top 10
df_top = df_merge.nlargest(10, 'cuoc_tt_tong')
```
