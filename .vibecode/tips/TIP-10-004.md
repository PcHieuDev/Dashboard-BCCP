## Yêu cầu
Thêm ROW mới dưới KPI cards hiện có: 3 thẻ sức khỏe KH.

### Card 1 — Gauge Retention Rate
```python
fig_gauge = go.Figure(go.Indicator(
    mode="gauge+number+delta",
    value=retention_rate,  # Từ get_retention_stats()
    domain={'x': [0, 1], 'y': [0, 1]},
    title={'text': "Tỷ lệ duy trì KH (%)"},
    delta={'reference': prev_retention_rate},
    gauge={
        'axis': {'range': [0, 100]},
        'bar': {'color': "darkblue"},
        'steps': [
            {'range': [0, 50], 'color': "#ffcccc"},
            {'range': [50, 80], 'color': "#fff3cd"},
            {'range': [80, 100], 'color': "#d4edda"}
        ],
        'threshold': {'line': {'color': "red", 'width': 4}, 'thickness': 0.75, 'value': 90}
    }
))
```
Data: `analytics/retention_metrics.py` → `get_retention_stats(db, year, month, cum)`

### Card 2 — KHM/Tái bán phát sinh
- Giá trị: SL mã CMS bán mới trong tháng (từ bảng `new_customers`)
- Delta: ± so tháng trước
- Sub: Tổng DT bán mới

### Card 3 — % DT Vãng lai
- Giá trị: `DT_vanglai / DT_tong × 100`
- Ngưỡng: `≤30%` ✅ Xanh | `>30%` ⚠️ Đỏ + icon cảnh báo
- Data: `query_revenue()` → group by loại KH → lọc "Vãng lai"

## Acceptance Criteria
- Gauge hiện Retention Rate % với target 90% (đường đỏ)
- Card KHM hiện SL + DT + delta so tháng trước
