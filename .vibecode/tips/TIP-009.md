## TASK

### Bước 1: Tạo `scripts/migrate_add_phbc.py` [NEW FILE]

Script tạo bảng `transactions_phbc` nếu chưa có trong DB:

```python
# scripts/migrate_add_phbc.py
import sqlite3
from pathlib import Path
import sys

project_root = Path(__file__).resolve().parent.parent
sys.path.insert(0, str(project_root))
from config.settings import DB_PATH

def migrate():
    conn = sqlite3.connect(str(DB_PATH))
    try:
        conn.execute("""
            CREATE TABLE IF NOT EXISTS transactions_phbc (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                thang_du_lieu TEXT NOT NULL,
                nam_du_lieu INTEGER NOT NULL,
                ma_buu_cuc TEXT NOT NULL,
                doanh_thu REAL DEFAULT 0,
                import_batch TEXT,
                created_at TEXT DEFAULT (datetime('now','localtime'))
            )
        """)
        conn.execute("""
            CREATE UNIQUE INDEX IF NOT EXISTS idx_phbc_unique
            ON transactions_phbc (thang_du_lieu, nam_du_lieu, ma_buu_cuc)
        """)
        conn.commit()
        print("[DONE] Bảng transactions_phbc đã sẵn sàng.")
    except Exception as e:
        print(f"[ERROR] {e}")
        conn.rollback()
    finally:
        conn.close()

if __name__ == "__main__":
    migrate()
```

**Chạy script ngay sau khi tạo** để tạo bảng trong DB thực.

### Bước 2: Thêm hàm import PHBC vào `etl/importer.py`

Thêm hàm `import_phbc_excel(db_path, excel_path, import_batch=None)` với logic:
1. Đọc file Excel (`.xlsx` hoặc `.xls`) bằng pandas
2. Chuẩn hóa cột:
   - Cột bưu cục → `ma_buu_cuc` (strip, upper)
   - Cột doanh thu → `doanh_thu` (float)
   - Cột tháng → `thang_du_lieu` dạng `T01`, `T02`... (từ cột tháng số nguyên hoặc string)
   - Cột năm → `nam_du_lieu` (int)
3. INSERT OR REPLACE vào `transactions_phbc`
4. Ghi log vào `import_log`
5. Trả về dict: `{"success": True, "rows": N, "message": "..."}`

**Template cột Excel PHBC** (phải detect tự động — không hardcode tên cột):
- Cột chứa `buu_cuc` hoặc `ma_bc` → `ma_buu_cuc`
- Cột chứa `doanh_thu` hoặc `dt` → `doanh_thu`
- Cột chứa `thang` → `thang_du_lieu`
- Cột chứa `nam` hoặc `year` → `nam_du_lieu`

## SPECIFICATIONS
- Bảng `transactions_phbc`: `id, thang_du_lieu(TEXT), nam_du_lieu(INT), ma_buu_cuc(TEXT), doanh_thu(REAL), import_batch(TEXT), created_at(TEXT)`
- UNIQUE INDEX trên `(thang_du_lieu, nam_du_lieu, ma_buu_cuc)` — không trùng lặp
- Import dùng `INSERT OR REPLACE` để cập nhật nếu đã có
- Nếu bảng đã tồn tại → script không làm gì, không lỗi
- Hàm import phải handle cả `.xls` và `.xlsx`
