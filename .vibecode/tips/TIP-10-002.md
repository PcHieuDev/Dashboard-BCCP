## Yêu cầu
Thêm **Annotated Heatmap** full-width dưới Stacked Bar. 18 Cụm (rows) × 4 Nhóm DV (cols).
- Giá trị: % tăng trưởng so **tháng trước** (default) hoặc **cùng kỳ YoY** (tùy chọn `sidebar-compare-mode`)
- Màu: RdYlGn (Đỏ=giảm, Vàng=0%, Xanh=tăng), `zmid=0`

### Callback: Thêm Output `global-heatmap`
```python
import plotly.figure_factory as ff

# Build heatmap
z = df_growth[["BCCP", "HCC", "TCBC", "PPBL"]].values.tolist()
x = ["BCCP", "HCC", "TCBC", "PPBL"]
y = df_growth.index.tolist()  # 18 Cụm

annotation_text = [[f"{val:+.1f}%" if val is not None else "N/A" for val in row] for row in z]

fig_heatmap = ff.create_annotated_heatmap(
    z=z, x=x, y=y,
    colorscale='RdYlGn',
    annotation_text=annotation_text,
    font_colors=['black', 'black'],
    zmid=0
)
fig_heatmap.update_layout(height=500, margin=dict(l=120))
```

### Tùy chọn so sánh
- Dùng `sidebar-compare-mode` (State) từ sidebar
- Nếu `compare_mode` chứa "yoy" → dùng YoY, else → tháng trước

## Acceptance Criteria
- Heatmap hiện 18 Cụm × 4 DV
- Ô xanh = tăng, ô đỏ = giảm, annotation hiện `+12.5%` hoặc `-3.2%`
- Default: so tháng trước. Khi chọn compare-mode "yoy" → so cùng kỳ
- Cụm có DT=0 tháng trước → hiện "N/A"
