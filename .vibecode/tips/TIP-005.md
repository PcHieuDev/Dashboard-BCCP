  - `scripts/sync_mappings.py` — Script cần gọi từ web

## TASK
1. **Sửa `dash_app/pages/import_data.py`**:
   - Thêm 1 khu vực mới ở **cuối trang** (sau bảng Lịch sử Import), gồm:
     - Tiêu đề: "📋 Quản lý Danh mục Dịch vụ"
     - `dcc.Upload(id="upload-csv-mapping", ...)` cho phép kéo thả file `.csv`.
     - `html.Div(id="csv-file-info")` hiển thị tên file đã chọn.
     - `dbc.Button("🔄 Đồng bộ danh mục Dịch vụ", id="btn-sync-mapping", color="warning")`.
     - `html.Div(id="sync-status-message")` hiển thị kết quả (dbc.Alert).

2. **Sửa `dash_app/callbacks/import_callbacks.py`**:
   - Callback 1: Khi user upload CSV → Lưu file vào `data/mapping-spdv.csv` (ghi đè). Hiển thị tên file + số dòng trong `csv-file-info`.
   - Callback 2: Khi user bấm "Đồng bộ" → Gọi `subprocess.run([sys.executable, "scripts/sync_mappings.py"], capture_output=True, text=True, cwd=project_root)`. Hiển thị kết quả (thành công/lỗi) trong `sync-status-message` bằng `dbc.Alert`.

## SPECIFICATIONS
- Chỉ chấp nhận file `.csv` (validate extension).
- File CSV phải có header chứa cột `nhom_chinh` (validate).
- Nút "Đồng bộ" chỉ active khi đã upload CSV thành công.
- Sau khi đồng bộ thành công, hiện Alert xanh. Lỗi → Alert đỏ kèm stderr.
- subprocess.run phải set `timeout=30` để tránh treo.
