## TASK

### Bước 1: Sửa `dash_app/components/sidebar.py`

**Cấu trúc sidebar mới** — 4 nhóm bộ lọc thay vì layout hiện tại:

```
📅 Nhóm 1: Thời gian (luôn hiện)
   - Năm, Chu kỳ (Ngày/Tuần/Tháng), Ngày/Tuần/Tháng picker

📊 Nhóm 2: So sánh (luôn hiện) — đặt ngay sau Thời gian
   - Checkbox: Kỳ trước, Cùng kỳ, Kế hoạch

🗺️ Nhóm 3: Bộ lọc chiều (luôn hiện)
   - Cụm, BĐX (Bưu điện huyện/xã), Bưu cục

📋 Nhóm 4: Bộ lọc dịch vụ BCCP (CHỈ hiện khi ở /bccp*)
   - Nhóm dịch vụ (sidebar-nhom-dv)
   - Loại khách hàng (sidebar-loai-kh)
   - Hợp đồng (sidebar-hop-dong)
   - ID container: "bccp-extra-filters" style có display:none mặc định
```

**Quan trọng**: Xóa hoàn toàn `sidebar-spdv` (Sản phẩm dịch vụ chi tiết) khỏi sidebar.

### Bước 2: Sửa `dash_app/callbacks/sidebar_callbacks.py`

Thêm Output mới vào callback `update_sidebar_state`:
```python
Output("bccp-extra-filters", "style"),  # Thêm vào cuối danh sách Output
```

Thêm logic trong hàm `update_sidebar_state`:
```python
# Hiện bộ lọc BCCP khi ở trang /bccp*
bccp_filter_style = {"display": "block"} if (pathname and pathname.startswith("/bccp")) else {"display": "none"}

# Thêm bccp_filter_style vào cuối tuple return
return (...existing outputs..., bccp_filter_style)
```

## SPECIFICATIONS
- `bccp-extra-filters` dùng CSS `display: block/none` — đơn giản, không animation
- `sidebar-spdv` bị xóa hoàn toàn khỏi DOM — không còn trong layout
- Nhóm So sánh di chuyển lên ngay sau Nhóm Thời gian (trước Địa lý)
- Khi ở `/hcc`, `/tcbc`, `/ppbl`, `/import`, `/` → Nhóm 4 ẩn hoàn toàn
