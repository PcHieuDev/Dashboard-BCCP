# TIP-10-008: Thêm Dropdown Mã DV vào /bccp/customer

## Header
- **TIP-ID**: TIP-10-008
- **Trang**: `/bccp/customer`
- **Files**: `pages/customer_detail.py`, `callbacks/customer_callbacks.py`
- **Priority**: P2 | **Effort**: Small

## Yêu cầu
Thêm dropdown **Mã dịch vụ cụ thể** (multi-select) vào Inline Filters, cho phép lọc theo mã SPDV chi tiết (VD: "N11 - CPN nội tỉnh").

## Chi tiết

### Layout: `pages/customer_detail.py`
Thêm dropdown sau `customer-filter-nhom-dv`:
```python
html.Div([
    html.Label("Dịch vụ cụ thể", className="filter-label"),
    dcc.Dropdown(
        id="customer-filter-spdv",
        multi=True,
        placeholder="Tất cả dịch vụ...",
        style={"fontSize": "13px"}
    )
], className="mb-2")
```

### Cascade: Nhóm DV → SPDV
Thêm callback cascade trong `customer_callbacks.py`:
```python
@app.callback(
    Output("customer-filter-spdv", "options"),
    [Input("customer-filter-nhom-dv", "value")]
)
def update_spdv_options(nhom_dv_selected):
    """Load SPDV options dựa trên nhóm DV đã chọn."""
    query = "SELECT ma_dich_vu, ten_dich_vu FROM dim_dichvu WHERE nhom_chinh = 'BCCP'"
    if nhom_dv_selected:
        placeholders = ",".join(["?"] * len(nhom_dv_selected))
        query += f" AND nhom_dich_vu IN ({placeholders})"
    # Return options: [{"label": "N11 - CPN nội tỉnh", "value": "N11"}, ...]
```

### Truyền filter vào query
Thêm `spdv` vào `State` list của 2 callbacks chính:
```python
State("customer-filter-spdv", "value")  # Thêm vào cuối State list
```
Truyền vào `resolve_filters_and_query()` → `query_revenue()` → filter `AND t.san_pham_dv IN (...)`

## Acceptance Criteria
- Dropdown SPDV cascade theo nhóm DV đã chọn
- Chọn SPDV → bảng DT và bảng CMS filter đúng
- Bỏ trống → hiện tất cả (không filter)
